This automation suite is for automation of 1 GET and 1 POST HTTPS request for almosafer.com/en which are:

Get request: https://www.almosafer.com/api/flight/resource/codes/DXB,JED
Post request: https://www.almosafer.com/api/v3/flights/flight/get-fares-calender


The framework is built using Cucumber with RestAssured and Junit on JDK 11

To run this project you will need following installed
- Java 1.8+ installed (Does not work with Java below 1.8)
- Maven Installed (I use version 3.8.1)
- Eclipse (Or another Java IDE)
